* Se tiver outras perguntas, [experimente perguntar à Comunidade do Power BI](http://community.powerbi.com/).
* Ainda tem problemas? Consulte a [página de suporte do Power BI](https://powerbi.microsoft.com/support/).

## <a name="power-bi-premium"></a>Power BI Premium
**O que é o Power BI Premium?**  
O Power BI Premium é uma nova oferta com base na capacidade, que inclui:

* A flexibilidade para publicar relatórios amplamente numa empresa, sem necessidade de os destinatários estarem licenciados individualmente por utilizador.
* Maior dimensionamento e desempenho da capacidade dedicada no serviço Power BI.
* A capacidade de manter recursos de BI no local com o Power BI Report Server.
* Um superfície de API, um conjunto consistente de capacidades e acesso às funcionalidades mais recentes para análise incorporada.

**O que faz o Power BI Premium? Como funciona?**  
O Power BI Premium consiste na capacidade no serviço Power BI exclusivamente atribuída a cada organização e suportada pelo hardware dedicado, totalmente gerido pela Microsoft. As organizações podem optar por aplicar a respetiva capacidade dedicada amplamente ou alocá-la às áreas de trabalho atribuídas com base no número de utilizadores, necessidades de carga de trabalho ou outros fatores — e aumentar ou reduzir verticalmente, à medida que os requisitos mudam.

**Em que é o que Power BI Pro diverge do Power BI Premium? Por que motivo o Power BI Premium é necessário?**  
O Power BI Premium é uma licença com base na capacidade, enquanto o Power BI Pro é uma licença baseada no utilizador.

O Power BI Premium consiste na capacidade do serviço Power BI exclusivamente atribuída a cada organização e suportada pelo hardware dedicado, totalmente gerido pela Microsoft — removendo as restrições de tamanhos de conjuntos de dados e taxas de atualização, e suportando o isolamento devido ao hardware dedicado. O Power BI Premium também proporciona a flexibilidade para publicar relatórios amplamente numa empresa, sem necessidade de os destinatários estarem licenciados individualmente por utilizador.

O Power BI Pro serve para os utilizadores publicarem relatórios, partilharem dashboards, colaborarem com colegas em áreas de trabalho e participarem noutras atividades relacionadas – como a capacidade de:

* Editar e guardar vistas personalizadas
* Criar dashboards pessoais (afixar ao novo dashboard)
* Analisar dados no Excel ou Power BI Desktop
* Partilhar com suporte para o Excel Web App
* Partilhar dashboards e colaborar com grupos do Office 365
* Integrar conteúdos com Microsoft Teams

**Preciso do Power BI Pro para utilizar o Power BI Premium?**  
Sim. O Power BI Pro é necessário para publicar relatórios, partilhar dashboards, colaborar com colegas em áreas de trabalho e participar noutras atividades relacionadas.

**Qual será o custo do Power BI Premium? Quantas SKUs serão disponibilizadas?**  
O Power BI Premium pode ser adquirido com base no número de núcleos virtuais. Pode ver os preços na [página de preços do Power BI](https://powerbi.microsoft.com/pricing/). Para obter mais informações sobre nós e núcleos virtuais, veja o [documento técnico do Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper). Utilize também esta [calculadora](https://powerbi.microsoft.com/calculator/) para obter uma estimativa da capacidade Premium que poderá ser necessária.

**O que significa "capacidade"?**  
"Capacidade" refere-se a uma capacidade nomeada, aprovisionada por um administrador através do portal de administração de capacidades do Power BI Premium. Uma capacidade é um agrupamento de um ou mais nós. Veja o [documento técnico do Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper) para obter mais informações.

**Como é faturado o Power BI Premium?**  
O Power BI Premium é faturado mensalmente, com um compromisso anual.

**Como irei comprar o Power BI Premium?**  
O Power BI Premium está disponível como um suplemento para o Power BI Pro no centro de administração do Office 365. Para obter mais informações, veja [Como comprar o Power BI Premium](service-admin-premium-purchase.md). Também pode contactar o representante da Microsoft para obter mais informações.

**Como posso saber quanta capacidade do Power BI Premium será necessário comprar?**  
Utilize esta [calculadora](https://powerbi.microsoft.com/calculator/) de custos para calcular a capacidade do Power BI Premium que poderá ser necessária.

**Quando ficará disponível o Power BI Premium? Estará disponível simultaneamente em todas as localizações geográficas?**  
O Power BI Premium será disponibilizado ao público em geral no final do segundo trimestre de 2017 para todas as localizações geográficas suportadas atualmente pelo serviço Power BI.

**O Power BI Premium irá cumprir a conformidade de certificação e regulamentar no momento da disponibilização comercial? E as clouds nacionais?**  
O Power BI Premium será disponibilizado ao público em geral no final do segundo trimestre de 2017 para todas as localizações geográficas suportadas atualmente pelo serviço Power BI. Planeia-se que as clouds nacionais — especificamente na Alemanha, China e Administração Pública dos Estados Unidos — fiquem disponíveis no terceiro trimestre de 2017. As normas de conformidade de certificação e regulamentares do Power BI aplicar-se-ão ao Power BI Premium no momento da disponibilização geral.

**O Power BI Premium está disponível com o Office 365 E5?**  
O Power BI Premium está disponível como um suplemento para o Power BI Pro. O Office 365 E5 inclui o Power BI Pro. Os clientes de E5 podem comprar o Premium como um suplemento para as respetivas licenças Pro existentes.

**O que é o Power BI Report Server? Isto significa que o Power BI fica disponível no local?**  
O Power BI Report Server está disponível como parte do Power BI Premium e é compatível com os relatórios do SQL Server Reporting Services, permitindo a implementação e distribuição de relatórios interativos do Power BI e relatórios paginados, no local dentro dos limites da firewall da organização. As organizações podem escolher o Power BI na cloud ou optar por manter relatórios no local com o Power BI Report Server e passar para a cloud ao seu ritmo.

As capacidades do Power BI, como dashboards em tempo real, consulta em idioma natural, etc., no serviço Power BI não estão disponíveis como parte do Power BI Report Server. 

Para obter mais informações sobre o Power BI Report Server, veja [Introdução ao Power BI Report Server](report-server/get-started.md).

**O Power BI Report Server é a funcionalidade que a Microsoft disponibilizou como pré-visualização técnica dos Relatórios do Power BI para SSRS? Por que motivo faz agora parte do Power BI, quando parecia ser uma funcionalidade do SQL?**  
O Power BI Report Server está disponível como parte do Power BI Premium e é compatível com os relatórios do SQL Server Reporting Services, permitindo a implementação e distribuição de relatórios interativos do Power BI e relatórios paginados, no local dentro dos limites da firewall da organização.

No anúncio do Power BI Premium divulgámos pela primeira vez como esta funcionalidade será lançada no mercado.

Para obter mais informações sobre o Power BI Report Server, veja [Introdução ao Power BI Report Server](report-server/get-started.md).

**O que são as Aplicações do Power BI?**  
Estamos a desenvolver pacotes de conteúdos em aplicações do Power BI, para melhorar a forma como os utilizadores descobrem e exploram informações à escala empresarial. Disponíveis atualmente, as aplicações do Power BI oferecem uma forma simplificada de implementar dashboards e relatórios para pessoas específicas, grupos ou toda a organização. Os utilizadores empresariais podem facilmente instalar estas aplicações e navegar nelas com facilidade, centralizando o conteúdo num único local e atualizando automaticamente. 

Para obter mais informações sobre as Aplicações, veja [O que são as aplicações do Power BI?](service-install-use-apps.md).

**Estão a efetuar alterações no Power BI Pro, Power BI Free, Power BI Desktop, etc., como parte da implementação do Power BI Premium?**  
Atualmente, também estamos a simplificar a distinção entre o Power BI Pro e o serviço gratuito. Recebemos comentários que, apesar de o serviço gratuito se destinar à utilização pessoal e o Power BI Pro permitir colaboração, as diferenças funcionais entre os dois gerou alguma confusão entre os utilizadores. Doravante, iremos melhorar o serviço gratuito para que tenha a mesma funcionalidade do Power BI Pro, mas iremos limitar as funcionalidades de partilha e colaboração apenas aos utilizadores do Power BI Pro.

Os utilizadores do serviço Power BI gratuito irão beneficiar do acesso a todas as origens de dados, aumento dos limites de armazenamento da área de trabalho e taxas de atualização e transmissão em fluxo superiores. Estas alterações entrarão em vigor em 1 de junho de 2017 e pode ler mais na [Inscrição de gestão personalizada no Power BI](service-self-service-signup-for-power-bi.md). O Power BI Desktop permanece disponível gratuitamente.

Para obter informações relativas à oferta da Versão de Avaliação Pro Expandida, veja [Ativação da Versão de Avaliação Pro Expandida](service-extended-pro-trial.md).

## <a name="power-bi-free"></a>Power BI Gratuito
**Que capacidades estão a ser adicionadas ao serviço gratuito?**  
A partir de 1 de junho de 2017, o serviço gratuito terá capacidades equivalentes ao Power BI Pro, exceto para a partilha, distribuição empresarial e colaboração. Isto inclui o mesmo limite da área de trabalho de 10 GB, até 8 atualizações diárias agendadas para conjuntos de dados e o máximo de 1M/hora de velocidade de dados de transmissão em fluxo. Estamos também a incluir acesso a todas as origens de dados, incluindo as que estão disponíveis através do gateway de dados no local.

**Quais capacidades do Power BI Pro não estão disponíveis no serviço gratuito?**  
A partilha do dashboard ponto-a-ponto, as áreas de trabalho de aplicações (anteriormente denominadas áreas de trabalho de grupo), Analisar no Excel e Aplicações do Power BI (a não ser que sejam suportadas pela capacidade Premium) são capacidades limitadas ao Power BI Pro.

**Quando entrarão em vigor estas alterações?**  
As alterações do serviço gratuito entrarão em vigor no dia 1 de junho de 2017.

**Como serão notificados destas alterações os utilizadores do serviço gratuito afetados?**  
Os utilizadores ativos do serviço gratuito afetados pelas alterações do dia 1 de junho de 2017 irão receber uma comunicação por e-mail da Microsoft, bem como notificações quando iniciarem sessão no serviço Power BI a partir de 3 de maio de 2017.

Os utilizadores existentes do serviço gratuito que estiveram ativos no último ano podem tirar partido da versão de avaliação gratuita expandida de 12 meses do Power BI Pro. A oferta irá fornecer as funcionalidades completas do Power BI Pro para garantir que os utilizadores têm tempo suficiente para ajustarem a forma como utilizam o serviço. Para obter informações relativas à oferta da Versão de Avaliação Pro Expandida, veja [Ativação da Versão de Avaliação Pro Expandida](service-extended-pro-trial.md).

**Os utilizadores perderão o acesso aos respetivos conteúdos (dashboards, relatórios) nessa altura?**  
Não. Não irão perder o acesso a quaisquer conteúdos que tiverem carregado para o serviço Power BI. Os conteúdos partilhados com outros utilizadores poderão já não estar acessíveis aos destinatários em 1 de junho de 2017. Da mesma forma, nessa altura podem já não ter acesso aos conteúdos que outros utilizadores partilharam com eles.

Os utilizadores existentes do serviço gratuito que estiveram ativos no último ano podem tirar partido da versão de avaliação gratuita expandida de 12 meses do Power BI Pro. A oferta irá fornecer as funcionalidades completas do Power BI Pro para garantir que os utilizadores têm tempo suficiente para ajustarem a forma como utilizam o serviço. Para obter informações relativas à oferta da Versão de Avaliação Pro Expandida, veja [Ativação da Versão de Avaliação Pro Expandida](service-extended-pro-trial.md).

**Como os utilizadores sabem se são elegíveis para a versão de avaliação do Power BI Pro expandida?**  
Consulte os [termos e condições](https://aka.ms/power-bi-trial) da Versão de Avaliação Pro Expandida.

Para obter informações relativas à oferta da Versão de Avaliação Pro Expandida, veja [Ativação da Versão de Avaliação Pro Expandida](service-extended-pro-trial.md).

**O que devem fazer os utilizadores se não forem elegíveis para a versão de avaliação do Power BI Pro expandida?**  
Inscreva-se numa versão de avaliação do Power BI Pro de 60 dias padrão [aqui](https://app.powerbi.com/signupredirect?pbi_source=web).

**A introdução do Power BI Premium altera as funcionalidades do serviço gratuito?**  
Os utilizadores do serviço gratuito com acesso à capacidade dedicada do Power BI Premium terão a capacidade de receber conteúdo que lhes foi distribuído por utilizadores do Power BI Pro através de Aplicações publicadas.

## <a name="power-bi-pro"></a>Power BI Pro
**O que muda no Power BI Pro?**  
Não estão previstas alterações ao Power BI Pro.

**E no Office – o Power BI Pro continua incluído no Office 365 E5?**  
Sim. O Power BI Pro continuará a ser incluído no Office 365 E5.

**É-me dada a opção de devolver as minhas licenças do Power BI Pro, se pretender mudar para o Power BI Premium?**  
Não. O Power BI Premium é um suplemento do Power BI Pro; por conseguinte, o Premium requer licenças Pro.

**Por que motivo o Power BI Pro é necessário? Não deveria apenas comprar o Power BI Premium?**  
O Power BI Premium é uma licença com base na capacidade, enquanto o Power BI Pro é uma licença baseada no utilizador, que se destina a uma análise do serviço de gestão personalizada fácil de utilizar, para os utilizadores que necessitam de colaboração, partilha de dashboard e análise ad hoc, e publicação de relatórios – como a capacidade de:

* Editar e guardar vistas personalizadas
* Criar dashboards pessoais (afixar ao novo dashboard)
* Analisar dados no Excel ou Power BI Desktop
* Partilhar e ver os livros do Excel com suporte para o Excel Web App
* Partilhar dashboards e colaborar com Grupos do Office 365
* Integrar conteúdos com Microsoft Teams

## <a name="power-bi-embedded"></a>Power BI Embedded
**O que mudou no Power BI Embedded?**  
Como parte do Power BI Premium, estamos a convergir o Power BI Embedded com o serviço Power BI para fornecer uma superfície de API, um conjunto consistente de capacidades e acesso às funcionalidades mais recentes. Daqui em diante, incentivamos os utilizadores interessados em incorporar o Power BI nas suas aplicações a começarem pelo Power BI Desktop e passarem para a implementação com o Power BI Premium.

**Quais são os benefícios do Power BI Premium para incorporar análise em aplicações?**  
O Power BI Premium é uma oferta com base na capacidade, que é executado em hardware dedicado e é totalmente gerido pela Microsoft como parte do serviço Power BI. Considere o ciclo de vida de desenvolvimento da sua aplicação, no qual o Power BI Premium irá oferecer novos benefícios, como custos flexíveis e previsíveis, gestão de conteúdos melhorada para os programadores e a disponibilização completa das funcionalidades do Power BI para cenários incorporados.

**É necessário migrar a minha aplicação existente incorporada do Power BI Embedded?**  
O Power BI Embedded irá continuar disponível para as aplicações existentes, de acordo com o seguinte agendamento:

| Contrato de licenciamento | Disponibilidade do Power BI Embedded |
| --- | --- |
| Contrato Enterprise |Até à expiração de contrato existente |
| Direct e CSP |Um ano de Disponibilidade Geral do Power BI Premium |

Sim, deve começar a considerar migrar a sua solução de análise incorporada para o Power BI Premium. Para obter mais informações, veja [Como migrar conteúdos da coleção de áreas de trabalho do Power BI Embedded para o Power BI](developer/migrate-from-powerbi-embedded.md).

**Como devo proceder para migrar a minha aplicação existente incorporada do Power BI Embedded?**  
Consulte a [documentação de migração](developer/migrate-from-powerbi-embedded.md) para obter orientações sobre a migração de aplicações.

**A Microsoft está a voltar atrás relativamente ao compromisso do Power BI Embedded Gratuito?**  
No modelo anterior, oferecíamos um número limitado de sessões para desenvolver e testar. No novo cenário, os clientes continuarão a ter opções disponíveis para começarem a desenvolver e testar gratuitamente, à sua maneira, para ficarem prontos para produção. Os clientes beneficiarão de uma versão de avaliação do Power BI Pro de 2 meses para começarem o desenvolvimento e teste. O Power BI Desktop, a ferramenta de exploração de dados e criação de relatórios, irá continuar a estar disponível gratuitamente.

**O Power BI Embedded continua a ser um Serviço do Azure? Onde posso comprar, aprovisionar e gerir o Power BI Embedded?**  
O Power BI Embedded está a ser convergido com o serviço Power BI e, por conseguinte, irá ser adquirido, aprovisionado e gerido como tal daqui em diante. Para obter mais informações, veja [Incorporar no Power BI](developer/embedding.md).

## <a name="power-bi-report-server"></a>Power BI Report Server
**Por que motivo o Power BI Report Server está incluído no Power BI Premium, em vez de ser vendido como um produto separado?**  
O Power BI Premium introduz a capacidade de manter recursos de BI no local com o Power BI Report Server. O Power BI Report Server é um servidor no local que permite a implementação e distribuição de relatórios interativos do Power BI e relatórios paginados, completamente dentro dos limites da firewall da organização. Com o Power BI Premium, o mesmo número de núcleos virtuais que uma organização aprovisiona na cloud também pode ser implementado no local através do Power BI Report Server, sem a necessidade de dividir a capacidade. As organizações podem escolher o Power BI na cloud ou optar por manter relatórios no local com o Power BI Report Server e passar para a cloud ao seu ritmo.

Para obter mais informações, veja [Introdução ao Power BI Report Server](report-server/get-started.md).

**Como podem os clientes obter acesso ao Power BI Report Server?**  
O Power BI Report Server está disponível através do Power BI Premium ou como um benefício para clientes com o SQL Server EE com SA. Contacte o representante da Microsoft para obter mais informações.

**Por que motivo o Power BI Report Server não está disponível como benefício para clientes do SQL Server Standard ou Enterprise Edition?**  
O Power BI Report Server está disponível para clientes do SQL Server com licenças EE + SA ou para uma compra através do Power BI Premium. O Power BI é um serviço SaaS, e o Power BI Report Server seguirá o mesmo modelo de entrega baseado na subscrição.
