Para obter o GUID do ID da área de trabalho, siga estes passos:

1. Inicie sessão no serviço Power BI.

2. Abra o relatório que quer incorporar.

3. Copie o GUID do URL. O GUID é o número entre **/groups/** e **/reports/** .
